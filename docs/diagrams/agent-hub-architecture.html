Current-state architecture

Agent Chat Hub

A Databricks Apps hosted multi-agent catalog and chat experience. React handles the catalog, chat shell, streamed responses, charts, and preferences; FastAPI coordinates OBO access, catalog discovery, chat dispatch, Lakebase state, and Databricks agent APIs.

Runtime Databricks Apps -> Uvicorn workers -> FastAPI `/api/v1` + built Vite UI Primary flow Catalog selection -> chat stream -> persisted conversation -> charts and suggestions
